# FlightAnomaly

FlightAnomaly is a flight anomaly detection platform for AIMS and MRO datasets.
It combines an unsupervised Isolation Forest pipeline with a supervised Random Forest model,
SQL Server connectivity via ODBC, and a Streamlit dashboard for result exploration.

## Key features

- `pipeline.py`: runs the full end-to-end workflow
  1. train Isolation Forest (`train.py`)
  2. detect IF anomalies on unmatched flights (`predict.py`)
  3. train supervised Random Forest (`train_rf.py`)
  4. scan RF to detect Isolation Forest false negatives
- `app.py`: main Streamlit app with multi-page navigation
- `config.py`: SQL Server settings, model paths, and feature definitions
- `db.py`: SQL Server access using SQLAlchemy + pyodbc
- `features.py`: flight data normalization and domain feature encoding

## Prerequisites

- Python 3.9+ (or compatible)
- Microsoft SQL Server with access to the following tables:
  - `AIMS`
  - `MRO`
  - `VolsValidesEtape1`
- ODBC driver for SQL Server (for example `ODBC Driver 17 for SQL Server`)

## Configuration

1. Open `config.py`.
2. Update `DB_CONFIG` with your SQL Server host, database, and driver.
3. Confirm the source table names if your schema differs.

## Usage

### 1. Install dependencies

```bash
python -m pip install -r requirements.txt
```

### 2. Run the full pipeline

```bash
python pipeline.py
```

This script runs the following stages:
- `train.py`
- `predict.py`
- `train_rf.py`
- final RF detection

### 3. Start the main web app

```bash
streamlit run app.py
```

## Output files

- `models/isolation_forest.pkl`: Isolation Forest model
- `models/random_forest.pkl`: Random Forest model
- `models/label_encoders.pkl`: feature encoder
- `logs/pipeline.log`: pipeline log
- `figures/`: plots produced by `train_rf.py`

## Notes

- `train.py` loads `VolsValidesEtape1` and filters on `EtapeValidation='1'` if the column exists.
- `train_rf.py` depends on `AnomaliesAIMS` and `AnomaliesMRO` generated by the IF pipeline.
- Domain features include route frequency, fleet family, and flight duration.
